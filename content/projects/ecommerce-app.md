## Descripción

Plataforma completa de e-commerce con frontend Angular, backend Node.js + Express, base de datos Mysql y pagos con Stripe.

## Stack

- **Frontend**: Angular 18, NgRx, Tailwind CSS
- **Backend**: Node.js, Express, TypeScript
- **Base de datos**: Mysql + DBeaver
- **Pagos**: Stripe Checkout
- **Deploy**: Docker Compose, Railway

## Funcionalidades

- Catálogo de productos con filtros y paginación
- Carrito persistente (localStorage + servidor)
- Checkout con Stripe
- Panel de administración para gestionar productos y pedidos
- Autenticación JWT con refresh tokens
